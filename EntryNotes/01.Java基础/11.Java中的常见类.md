# Properties类

- 一个数组中可以存放多个元素，多个元素必须是同一种类型，且数组长度一旦确定就不能改变。

- **集合**和数组一样也可以存放多个元素，并且允许元素是不同的类型，且集合的长度可以改变。

​     int[] arr = new int[10];

​     Java中提供类一些集合类，一个集合类的对象就是一个集合。
