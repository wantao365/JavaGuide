# HTML标签（功能标签）

- 显示标签：将文本、图片、声音和视频展示给用户

- 输入标签：给用户提供输入视图，用于输入数据

- 功能标签：没有视图效果，完成某种特定的功能

## 超链接

作用：当点击特定的本文/图片提示，可以**跳转**到指定的位置
